Tooling

  • Download Node.js from https://nodejs.org

    Node.js is an environment for executing JavaScript on the server (desktop).

    For windows, download the long term support release (LTS). The current version is 6.11.5

  • Download Visual Studio Code from https://code.visualstudio.com/

    A free, open-source code editor, which is easy to use and much lighter than Visual Studio.

    (Other editors are available)

  • Install the Angular CLI

    The Angular Command Line Interface (CLI) is software that runs on Node.js

    Node Package Manager is how you install it and all other Node.js software

    Run Node Package Manager (npm) on the command line:

    > npm install -g '@angular-cli'
  • Create a new project with the Angular CLI

    ... and run it.

    > ng new angular-zoo
> cd angular-zoo
> ng serve --open
